# Sound Change Applier (Rust)

**Sound Change Applier (Rust)** (aka `scars`) is a CLI tool for conlangers developing language families. It can quickly apply a set of sound change *rules* to an input *lexicon*, generating an output lexicon with the rules applied.

`scars` is based on the [Sound Change Applier](https://www.zompist.com/sca2.html) by zompist.
